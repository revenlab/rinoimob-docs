# Rinoimob Documentation

Technical documentation for the Rinoimob platform.

## Contents

- **Architecture** - System design and diagrams
- **API Reference** - Backend API endpoints and schemas
- **Frontend Guides** - Vue 3 and Nuxt 3 development
- **Deployment** - Infrastructure and deployment guides
- **Contributing** - Development guidelines and workflows

## Getting Started

### View Online
```bash
mkdocs serve
```

### Build Static Site
```bash
mkdocs build
```
